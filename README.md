# Discord Webhook Multitool

A simple Python tool that allows you to interact with Discord webhooks via an ASCII-based interface. The tool provides various functionalities such as sending spam messages, renaming webhooks, deleting them, changing their profile pictures, and exiting the program.

## Features

Spam Message: Send a specified number of messages to a webhook.
Rename Webhook: Change the name of an existing webhook.
Delete Webhook: Delete a specific webhook.
Change Webhook PFP: Update the profile picture (avatar) of a webhook.
Exit Program: Safely exit the tool.
